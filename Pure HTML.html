Pure HTML Practice Page

This page is written in pure HTML only. As a student, your task is to improve its appearance using CSS.

About this page

This webpage is intentionally simple because it serves as a starting point for CSS practice. The structure is already provided using pure HTML, and your goal is to make it visually better by applying the CSS properties that you have learned in class.

By styling this page, we will see how creative you can be when working with colors, font sizes, font families, font weights, text alignment, and other related CSS properties. The content is already here, but the design will depend on your creativity and understanding of CSS.

Topics Covered

  • Headings and paragraphs
  • Links
  • Images
  • Lists
  • Tables
  • Forms and input types

Styling Challenge

  1. Change the background color and text color.
  2. Choose a better font family for the page.
  3. Adjust the font size of headings and paragraphs.
  4. Improve the text alignment where necessary.
  5. Apply other related CSS properties to make the page more attractive.
